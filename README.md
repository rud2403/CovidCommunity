# Introduction

백신후기 공유 플랫폼


- 개발 기간 : 22.03.13 - 22.04.17
- 팀 구성 : 개인 프로젝트(1명)
- 주요 기능 : 회원 및 게시판 CRUD, 공공데이터 API, 소셜 로그인, SpringSecurity 기반의 인증 및 인가

- 형상 관리 : git/github

# Features

**User**
- 회원가입, 로그인(SpringSecurity)
- 소셜 로그인(구글)
- 마이 페이지

**Board**
- 후기 게시판
- 필터링(검색, 카테고리)
- Pagination
- 댓글(대댓글)
- 좋아요
- 이미지 파일 첨부
- 발생현황, 접종현황(공공데이터포털 API)

**Admin**
- 회원 관리
- 게시판 관리
